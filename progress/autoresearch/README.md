# Autoresearch progress area

This directory is the canonical home for append-only experiment logs produced by autoresearch lanes.

## Intended file

- `experiments.jsonl` — live append-only ledger (created during actual runs)

## Checked-in example

- `experiments.template.jsonl`

## Usage rule

Do not treat this directory as a place for narrative notes only. The main artifact should be machine-readable run history that agents and humans can audit.
