# Hurst exponent

A single number \(H\) summarizing **memory** in a monofractal picture:

- \(H \approx 0.5\): more **random-walk-like**
- \(H > 0.5\): **persistent** (trends tend to continue)
- \(H < 0.5\): **anti-persistent** (mean-reverting jitter)

Estimated via **R/S**, **[[DFA detrended fluctuation|DFA]]**, wavelets, etc.
